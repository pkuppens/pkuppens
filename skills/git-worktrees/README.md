# Git Worktrees Skill

Explains git worktrees and Claude Code usage. Use when a branch cannot be deleted because of a worktree, when cleaning up `.claude/worktrees/`, or when asked about worktrees.

## References

- [Git worktree documentation](https://git-scm.com/docs/git-worktree) — official Git reference
- [Claude Code worktree guide](https://claudelab.net/en/articles/claude-code/worktree-guide) — Claude Code worktree usage and parallel development
- [Claude Code git workflow automation](https://claudelab.net/en/articles/claude-code/claude-code-git-workflow-automation) — advanced branch management with Claude

## Usage

Invoke the skill when the user hits "cannot delete branch used by worktree", asks about worktrees, or needs to clean up Claude session worktrees. See [SKILL.md](SKILL.md) for the full procedure.
